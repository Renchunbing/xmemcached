**XMemcached** is a high performance, easy to use multithreaded memcached client in **java**.

It's **nio** based and was carefully turned to get top performance.

**Xmemcached 2.0.0 has been released! Please check [ReleaseNotes](ReleaseNotes.md).**

**Xmemcached has been moved to github: https://github.com/killme2008/xmemcached**

If you are using maven2 to build your project,you can try it now:
```
  <dependency>
      <groupId>com.googlecode.xmemcached</groupId>
      <artifactId>xmemcached</artifactId>
      <version>2.0.0</version>
 </dependency>
```

A clojure library [clj-xmemcached](https://github.com/killme2008/clj-xmemcached).
